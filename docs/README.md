项目文档（docs）

本目录包含面向开发者的简要文档，帮助理解项目结构、关键单例与关卡切换实现，以及常见问题的调试方法。

包含文件：
- `quick_start.md`：快速上手，如何打开与运行项目（Godot 版本、命令行示例）。
- `architecture.md`：项目总体架构与关键 Autoload（单例）说明。
- `level_transition.md`：关卡传送/切换实现细节（`LevelTransition` 与 `LevelManager` 的交互）。
- `debugging.md`：复现/调试“跳过场景”问题的步骤、建议日志点与快速修复建议。

如果需要我可以将文档翻译为英文或生成图片/时序图放到 `docs/assets/`。
